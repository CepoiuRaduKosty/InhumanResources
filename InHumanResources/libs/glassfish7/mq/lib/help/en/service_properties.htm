Service Properties

Use this dialog to specify the properties of the selected Message Queue service. You can change any of the following properties without pausing the service.

Port Number Specifies the number of the port that was dynamically allocated to the selected service by the port mapper. You can click the Static radio button and specify an alternate (available) port number for the service.

Service State Specifies the current state of the service.

Current Number of Allocated Threads Specifies the number of threads allocated to this service.

Current Number of Connections Specifies the number of connections currently allocated to this service.

Min Number of Threads Specify the minimum number of threads that should be allocated from the thread pool for this service.

Max Number of Threads Specify the maximum number of threads that should be allocated from the thread pool for this service.

Click OK to set these configuration values for the selected service.
